# Teste de Dev Backend usando Ruby!

O objetivo deste teste é avaliar a organização do seu código, e como você estrutura uma
solução simples e eficiente baseada em requisitos simples.

------------

#### Ruby version
```
2.4.0
```

------------

#### Database creation
```
rake db:create
rake db:migrate
```

------------

#### How to run the tests
```
rspec -f d
```
